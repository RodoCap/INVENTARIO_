#  Sistema de Inventario en Python

Proyecto desarrollado en Python utilizando arquitectura MVC (Modelo - Vista - Controlador) y base de datos SQLite para la gestión de productos en un inventario.

---

# 🚀 Funcionalidades

✅ Registrar productos  
✅ Mostrar productos  
✅ Actualizar stock  
✅ Eliminar productos  
✅ Buscar productos por categoría  
✅ Mostrar productos con bajo stock  
✅ Persistencia de datos con SQLite  

---

# 🛠️ Tecnologías utilizadas

- Python 3
- SQLite3
- Arquitectura MVC
